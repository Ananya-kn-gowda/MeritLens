# 🌟 MeritLens — Seeing the Talent, Not Just the Score

## 🚀 Problem

Traditional evaluation systems rely only on raw scores (GPA/test results).
This ignores the candidate’s background — making it unfair for students from low-resource environments.

---

## 💡 Solution

MeritLens introduces a **Grit Multiplier** that adjusts scores based on:

* School type
* Family income

This helps identify **“Hidden Stars”** — candidates who perform well despite challenges.

---

## 🧠 How It Works

1. User inputs:

   * Score
   * School Type
   * Family Income

2. System calculates:

   * Income multiplier
   * School bonus

3. Final:

   * Adjusted Score = Raw Score × Grit Multiplier

4. Generates insight:

   * Explains fairness and context
